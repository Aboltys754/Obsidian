Инициализировать новый проект в новый репозиторий. Пример репозиторий Git
```git
echo "# Git" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:Aboltys754/Git.git
git push -u origin main
```
