Проект работает на python3.8
1. Создаем и активируем виртуальное окружение
2. ```pip install --upgrade pip```
3. ```pip install "apache-airflow==2.1.3" --constraint "https://raw.githubusercontent.com/apache/airflow/constraints-2.1.3/constraints-3.8.txt"   ```
4. ```pip install openpyxl==3.1.5```
5. pip install -r requirements.txt
6.  Далее прописываем пусть к домашней директории 
export AIRFLOW_HOME='ТУТ ПИШЕШЬ СВОЙ ПУТЬ'
7. Инициализируем БД (для локального запуска использую дефолтный) sqlite
            airflow db init
8. Создаем юзера
     airflow users create \
      --username olga \
      --firstname Admin \
      --lastname User \
      --role Admin \
      --email admin@example.com \
      --password olga
   9. Запускаем веб-сервер
       airflow webserver --port 8080
10. Открываем еще один терминал
11. Активируем виртуальное окружение
12. Прописываем домашнюю директорию
13. Запускаем шедулер
      airflow scheduler
