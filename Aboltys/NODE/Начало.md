для импорта  http можно использовать commonJS  или ES6

```js
Это для commonJS
const http = require('http');
```

```js
это для ES6
import * as http from 'http';
```
объявление сервера и запуск на порт 3000

const http = require('http');
const server = new http.Server();

server.on('request', (req, res) => {
	res.end
});

server.listen(3000);
